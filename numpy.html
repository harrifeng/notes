1 多维数组

1.1 多维数组创建

  • 使用array来创建多维数组 

    >>> import numpy as np
>>> a = np.array([[0, 1, 2, 3], [10, 11, 12, 13]])
>>> print(a)
[[ 0  1  2  3]
 [10 11 12 13]]
>>> a.shape
(2, 4)
>>> a.size
8
>>> a.ndim
2

1.2 多为数组索引

  • 我们既可以使用python内置的[x][y]的方法进行索引 

    >>> import numpy as np
>>> a = np.array([[0, 1, 2, 3], [10, 11, 12, 13]])
>>> a[1][3]
13

  • 还可以使用numpy特有的索引方法,也就是"x,y"的方法索引,其实这里是省略了一个`()`其实应该是 

    >>> import numpy as np
>>> a = np.array([[0, 1, 2, 3], [10, 11, 12, 13]])
>>> a[1,3]
13
>>> a[(1,3)]
13

  • 我们还可以使用单个索引来索引一整行,注意,这里返回的不再是一个数字了,而是一个array 

    >>> import numpy as np
>>> a = np.array([[0, 1, 2, 3], [10, 11, 12, 13]])
array([10, 11, 12, 13])

1.3 多维数组切片

  • 多维数组也支持切片操作
  • 切片操作会定位一个matrix出来,定位的方法还是传入一个tuple(x1到x2行,y1到y2列),当然了,`()`会省略

  • 比如我们想获得第一行的第4个,第5个元素,就有 

    >>> import numpy as np
>>> a = np.array([[ 0, 1, 2, 3, 4, 5],
           [10,11,12,13,14,15],
           [20,21,22,23,24,25],
           [30,31,32,33,34,35],
           [40,41,42,43,44,45],
           [50,51,52,53,54,55]])
>>> a[0:1,3:5]
array([[3, 4]])

  • 上面的行数是一行,我们甚至可以把第一个"0:1"简化成"0", 但是我们可以看到返回值是一个一维数组(而不是上面的两维数组) 

    >>> a = np.array([[ 0, 1, 2, 3, 4, 5],
           [10,11,12,13,14,15],
           [20,21,22,23,24,25],
           [30,31,32,33,34,35],
           [40,41,42,43,44,45],
           [50,51,52,53,54,55]])
>>> a[0,3:5]
array([3, 4])

  • 我们的`:`符号当然是可以省略的了,比如我们可以省略一半,比如下面求整个matrix的最后两行的最后两列 

    >>> a = np.array([[ 0, 1, 2, 3, 4, 5],
           [10,11,12,13,14,15],
           [20,21,22,23,24,25],
           [30,31,32,33,34,35],
           [40,41,42,43,44,45],
           [50,51,52,53,54,55]])
>>> a[-2:,-2:]
array([[44, 45],
       [54, 55]])
>>> a[4:,4:]
array([[44, 45],
       [54, 55]])

  • 我们甚至可以把`:`的左右都省略掉(都省略就是0:length的意思),比如,我们求第3列,方法如下,注意得到的是一个二维数组 

    >>> a = np.array([[ 0, 1, 2, 3, 4, 5],
           [10,11,12,13,14,15],
           [20,21,22,23,24,25],
           [30,31,32,33,34,35],
           [40,41,42,43,44,45],
           [50,51,52,53,54,55]])
>>> a[:,2:4]
array([[ 2,  3],
       [12, 13],
       [22, 23],
       [32, 33],
       [42, 43],
       [52, 53]])
>>> a[0:6,2:4]
array([[ 2,  3],
       [12, 13],
       [22, 23],
       [32, 33],
       [42, 43],
       [52, 53]])

  • 如果我们想得到第三列,但是只想得到一个一维数组,那么我们可以把tuple的第二个成员赋成数字,而不是slice 

    >>> a = np.array([[ 0, 1, 2, 3, 4, 5],
           [10,11,12,13,14,15],
           [20,21,22,23,24,25],
           [30,31,32,33,34,35],
           [40,41,42,43,44,45],
           [50,51,52,53,54,55]])
>>> a[:,2]
array([ 2, 12, 22, 32, 42, 52])

  • 切片的step规则也是支持的([lower:upper:step]), 比如我们这里取出第3行,第5行的奇数列 

    >>> a = np.array([[ 0, 1, 2, 3, 4, 5],
           [10,11,12,13,14,15],
           [20,21,22,23,24,25],
           [30,31,32,33,34,35],
           [40,41,42,43,44,45],
           [50,51,52,53,54,55]])
>>> a[2::2, ::2]
array([[20, 22, 24],
       [40, 42, 44]])
